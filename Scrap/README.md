# XL Scrap

Adds XL Scrap using the `XL Scrap API` mod as an example.

### Compatibility

Works with v64 of Lethal Company, and is required by all clients.

### Current Scrap:

- CRT TV
- L Couch (Scrap finished but dysfunctional; API needs updating)
